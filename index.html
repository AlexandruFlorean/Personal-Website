JAVA TIC TAC TOE

The Tic-Tac-Toe game follows the MVC pattern to separate concerns between game logic, user interface, and user interactions. It features three gameplay modes: Player vs. Player, Player vs. Computer and Computer vs. Computer. The GUI is implemented using Java Swing, providing an interactive and user-friendly experience. The Player vs. Computer mode utilizes the Minimax algorithm, ensuring that the AI always makes the optimal move, making it impossible for a human player to win.
